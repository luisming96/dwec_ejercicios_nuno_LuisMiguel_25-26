T02 - Ejercicio11 - Luis Miguel Nuno

Realizar un programa en script que calcule el factorial impar de un número entero. El factorial impar de un número n es el producto de todos los números naturales impares desde el 1 hasta n o n-1, dependiendo de si n es par o impar.
Ejemplo: 5! = 5 x 3 x 1 = 15 10! = 9 x 7 x 5 x 3 x 1 = 945 La solución se debera de hacer de forma recursiva.
