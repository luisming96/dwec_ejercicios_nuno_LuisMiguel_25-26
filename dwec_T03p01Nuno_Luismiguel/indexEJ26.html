26.Elabora un script que pida al usuario una cadena y dos letras. Después reemplazará una letra por otra. Si la primera letra no existe se indicará un error. Debes resolverlo usando el método replace() del objeto String usando expresiones regulares. No debe distinguir entre mayúsculas y minúsculas. Controla esto desde la expresión regular.
