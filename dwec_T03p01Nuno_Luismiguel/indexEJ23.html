Deberás hacer uso del objeto ExpReg y crear una función que se denomine "validarFormatoFecha()" que reciba la cadena introducida por el usuario y devuelva un booleano. La expresión regular debes crear usando el método: Puedes usar una IA para generar el patrón, entendiendo dicho patrón. Después tienes que crear un objeto de tipo Date y determinar si la fecha es correcta. Objeto String y RegExp
